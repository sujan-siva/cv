Adding Images

img src="" alt=""

  • In this tag the src attribute is where the image file will be used it can be used as either a URL or a directory to the working folder
  • The alt attribute is text that will appear in place of the image if it is not possible to be shown for whatever reason.

Adding an Anchor Tag / External Link

  • Here the href is the link to external page or internal html page
  • The brackets are for putting the title of the link
